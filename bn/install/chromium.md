# ইনস্টল
### Chrome, Opera (GX), Edge, etc.
1. আমাদের ডাউনলোড পেজে যাও [এখানে ক্লিক করে](https://premid.app/downloads)।
2. Choose **Chrome**.
3. Click on **Add to Chrome**. (The text may differ. Look for a button that indicates installation of the extension.)
