安裝擴充功能非常重要，因為單靠本地程式是無法做任何事的。

# 安裝
1. [點此](https://premid.app/downloads)前往我們的下載頁面
2. 選擇 **FireFox**
3. 搜尋 `about:addons`
4. 點擊右上角的小齒輪圖標。
5. 選擇 **從檔案安裝擴充套件**
6. 選擇你剛剛下載的檔案
7. 點擊 **新增** 以確認載入

> 別忘了[安裝**安裝我們的本地應用程式**](/install)
